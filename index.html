🔧 Configurar Firebase

Para que cada usuario tenga su propio progreso guardado en la nube, necesitas conectar Firebase. Sigue estos pasos:

1Ve a console.firebase.google.com, crea un proyecto y llámalo panini-fwc2026
2En Authentication → Sign-in method, habilita Email/Contraseña y Google
3En Firestore Database, crea una base de datos en modo producción (o test para pruebas)
4En Configuración del proyecto → Aplicaciones web, registra una nueva app y copia el objeto firebaseConfig
5Abre index.html, busca la sección FIREBASE CONFIG y reemplaza los valores de firebaseConfig
6Sube el archivo a GitHub Pages y en Firebase → Authentication → Dominios autorizados, añade tu dominio usuario.github.io
// Reglas recomendadas para Firestore: rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /albums/{userId} { allow read, write: if request.auth.uid == userId; } } }
